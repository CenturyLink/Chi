Chi Playground

In order to change the theme, you can use the following routes:

  • Chi (default): http://localhost:5173
  • Portal: http://localhost:5173/?theme=chi-portal
  • Connect: http://localhost:5173/?theme=chi-connect
  • Centurylink: http://localhost:5173/?theme=chi-centurylink
  • Colt: http://localhost:5173/?theme=chi-colt
  • Brightspeed: http://localhost:5173/?theme=chi-brightspeed
  • Test: http://localhost:5173/?theme=chi-test
